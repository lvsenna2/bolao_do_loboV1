# Features

Feature modules are organized around product capabilities from the SDD.

Initial modules:

- `auth`
- `admin`
- `dashboard`
- `leagues`
- `matches`
- `guesses`
- `rounds`
- `ranking`
- `payments`
- `notifications`
- `profile`

Each feature may contain `components`, `actions`, `schemas`, `services`, `types` and
tests when the corresponding phase is implemented.
